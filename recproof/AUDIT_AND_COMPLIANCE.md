# Audit and Compliance

RecruitProof should log every important action in a proof run.

## Audit events

- Export received
- File hash created
- Parse completed
- Parse failed
- Duplicate cluster created
- Candidate viewed
- Search executed
- Shortlist exported
- Report downloaded
- User permission check

## Compliance posture

RecruitProof supports defensible recruiting operations by making search, ranking, and export actions traceable. It does not replace legal review of hiring, privacy, or employment obligations.
