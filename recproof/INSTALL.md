# Million CV Proof Dashboard Add-On

This drop-in package adds a Next.js enterprise demo shell to RecruitProof without changing the Python search engine.

## Install

Copy these files into the root of `Scrutexity/RecruitProof`, then run:

```bash
npm install
npm run dev
npm run lint
npm run agent-browser
```

## What it adds

- 12-section enterprise sidebar demo
- Executive dashboard
- Import center
- Intelligence dashboard
- Enterprise search experience
- Candidate intelligence page
- ROI dashboard
- Migration center
- Audit center
- Trust center
- Million-CV proof report
- Demo storyboard
- Docs copied into `public/docs`
