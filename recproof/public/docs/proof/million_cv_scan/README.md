# Million CV Scan Proof

This folder is the proof package for the RecruitProof enterprise demo. It is designed for a read-only Encore archive export of 500k-1M CV files and a synthetic fallback dataset when real data is not yet available.

## What this proves

- RecruitProof can ingest large CV archives without replacing the ATS first.
- RecruitProof can parse PDF and DOCX attachments into normalized candidate records.
- RecruitProof can identify hidden high-fit candidates already paid for by the recruiting firm.
- RecruitProof can benchmark latency, duplicate rate, failure rate, and ROI in an executive-ready report.

## Demo position

Do not claim these are Rudy's real numbers until the Encore export is processed. In the demo shell, all numbers are synthetic but shaped to mirror the production proof conversation.
